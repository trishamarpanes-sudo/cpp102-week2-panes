Laboratory Activity 2: Compute the Average of Three Quiz Scores
Step 1: Start
Step 2: Ask the user to enter the first quiz score. Store it as score1
Step 3: Ask the user to enter the second quiz score. Store it as score2
Step 4: Ask the user to enter the third quiz score. Store it as score3
Step 5: Compute the average = (score1 + score2 + score3) / 3
Step 6: Display the average
Step 7: End

Laboratory Activity 3: Determine if a Number is Even or Odd
Step 1: Start
Step 2: Ask the user to enter a number. Store it as num

   Display "The number is Even"
  Else
